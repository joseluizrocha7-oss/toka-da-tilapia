# Catálogo de skills — BaseOS

> Referência completa das skills disponíveis. A skill /mapear-rotinas lê esse arquivo
> antes de criar uma nova skill, pra ver se já existe cobertura.

---

## Núcleo — operação diária

| Skill | O que faz | Quando usar |
|-------|-----------|-------------|
| `/abrir` | Carrega o contexto da agência e abre a sessão | Início de cada sessão de trabalho |
| `/salvar` | Faz commit + push no GitHub | Quando quiser salvar o trabalho |
| `/atualizar` | Varre o workspace e atualiza os arquivos de contexto | Quando algo mudou e o contexto ficou desatualizado |
| `/instalar` | Setup inicial do BaseOS — entrevista e configura todos os arquivos | Primeira vez no repositório |
| `/mapear-rotinas` | Mapeia tarefas repetitivas e cria skills personalizadas | Quando quiser automatizar algo que repete toda semana |

---

## Clientes e projetos

| Skill | O que faz | Quando usar |
|-------|-----------|-------------|
| `/briefing-cliente` | Cria pasta estruturada + briefing completo de um novo cliente | Ao fechar ou prospectar um cliente |
| `/novo-projeto` | Cria pasta de projeto novo com CLAUDE.md dedicado | Projeto interno ou iniciativa da agência |
| `/pipeline` | Exibe e atualiza o funil comercial (leads → proposta → fechado) | Acompanhar o status de leads e clientes |
| `/proposta` | Cria proposta comercial profissional em HTML | Pra enviar orçamento pra lead |

## Conteúdo e SEO

| Skill | O que faz | Quando usar |
|-------|-----------|-------------|
| `/carrossel` | Cria carrossel ou post visual em HTML + PNG 1080x1350 | Qualquer conteúdo visual pro Instagram |
| `/publicar-tema` | Pipeline completo: blog + carrossel + legendas (Insta, FB, LinkedIn) | Quando tem um tema e quer publicar em todos os canais |
| `/aprovar-post` | Publica conteúdo já criado: flipa draft, commita, posta via Meta API | Quando o conteúdo foi revisado e está pronto pra ir pro ar |
| `/seo` | Fluxo completo de SEO + GEO + Google Ads em 8 passos | Pesquisa de demanda, análise de concorrência, GMB, on-page, ads |
| `/responder-avaliacoes` | Escreve respostas humanas pras reviews do Google | Quando tem avaliações novas no GMB |
